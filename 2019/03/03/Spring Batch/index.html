Spring Batch支持任务分发吗？

支持。随着业务量的增加，单个节点可能无法满足job的处理，需要采用远程Step的方式将多个节点组合起来完成Job的处理。

Remote chunking. 远程Step技术就是将对Item的读操作与写操作分离，读操作由一个节点来完成，写操作交由另外一个节点来完成。

在Master节点，作业步负责读取数据，并将读取到的数据通过远程技术发送给远端节点，进行处理，处理完成后Master节点负责回收远端节点的处理结果。

Spring Batch框架中提供了两个核心接口完成远程Step任务，ChunkProvider与ChunkProcessor。
