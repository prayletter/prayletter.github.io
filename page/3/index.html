tomcat类加载器

发表于 2019-03-03 | 阅读次数：
  • Common ClassLoader. Tomcat最基本的类加载器，加载/common/*路径下的Java类库，加载的类库可以被Tomcat容器本身与WebApp访问。
  • Catalina ClassLoader. Tomcat私有类加载器，加载路径中的class对WebApp不可见。
  • Shared ClassLoader. WebApp共享的类加载器，加载路径中的class对所有WebApp可见，对Tomcat容器不可见。
  • WebApp ClassLoader WebApp私有的类加载器，加载路径中的class只对当前WebApp可见。

MySQL如何应对高并发

发表于 2019-03-03 | 阅读次数：
  • 结合具体业务进行合理的分库分表。
  • 数据库集群。读写分离，避免DB宕机带来的单点DB不能访问的问题。
  • 适当增加冗余字段避免连表查询。
  • 使用InnoDB引擎，Myisam表级锁是致命问题。
  • 使用缓存。热点数据放入缓存。使用内存锁，锁主键，当一个线程修改某id的数据时，将该id存入缓存中，若其他线程触发修改该id的数据时，读到缓存中有该id，则阻止修改。

如何应对高并发

发表于 2019-03-03 | 阅读次数：
  • 尽量使用客户端浏览器缓存，减少请求服务器，例如js、css、图片等。
  • 使用CDN内容分发网络，提高大图片与静态页面的访问速度。
  • 增加服务器，做出集群，通过硬件与软件负载均衡，将请求分发到不同的服务器上。
  • 使用消息队列，前端的请求先放入队列中，直接返回用户成功信息，后端通过线程池进行异步处理。
  • 使用缓存。将热点数据缓存进内存数据库中，减轻后台数据库的压力。
  • 数据库索引。通过explain检查SQL性能，给查询字段增加索引，提高数据库查询效率。
  • 数据库分库分表，降低单库单表的数据量。
  • 数据库集群，读写分离。
